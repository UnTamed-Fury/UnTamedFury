# UNTAMED_FURY

My personal portfolio website built with Node.js and Express.

## Deployed on Netlify
[![Netlify Status](https://api.netlify.com/api/v1/badges/YOUR-SITE-ID/deploy-status)](https://app.netlify.com/sites/YOUR-SITE-NAME)

## Features
- Modern glass-morphism design
- Responsive layout
- Animated title effects
- About and Contact pages
- Social media links

## Technologies Used
- Node.js
- Express.js
- HTML5/CSS3
- JavaScript

## Local Development

1. Clone the repository
2. Install dependencies: `npm install`
3. Start the development server: `npm run dev`
4. Visit `http://localhost:3000`

## Deployment
This site is configured for deployment on Netlify. The build process will:
1. Install dependencies using `npm install`
2. Serve static files from the `public` directory
3. Run the Node.js server

## Project Structure
- `server.js`: Express server configuration
- `public/`: Static assets (HTML, CSS, JS, images)
- `package.json`: Dependencies and scripts
- `netlify.toml`: Netlify configuration
